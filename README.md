# tokimo-package-filetype-detector

Universal file type detector — binary/text detection, encoding, MIME type, and programming language identification.

## Features

| Feature | Description |
|---|---|
| Binary detection | Magic bytes + heuristics |
| Encoding detection | UTF-8, UTF-16, Shift-JIS, GBK, etc. |
| MIME detection | Content-type from file signatures |
| Language detection | Programming language from file extensions and content |

## Add to Cargo.toml

```toml
tokimo-package-filetype-detector = { git = "https://github.com/tokimo-lab/tokimo-package-filetype-detector" }
```
